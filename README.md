# Ignite Shop Challenge

In this challenge, I implemented a shopping cart that uses Stripe API data to search for existing items and controls the number of items a user wants to purchase.

## Stack

Next

React

## Run local

Instale com npm

```bash
  npm install
  npm run dev
```

or
```bash
  yarn install
  yarn dev
```
